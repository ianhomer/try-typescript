# tl;dr

## prepare

    cp .env.example .env
    
## run

    npm start

## test

    http http://localhost:4040/

# debug

Set break point in ts code and IntelliJ -> Run -> Debug 'local.ts'.

# build

    npm run build

and run local build

    npm run local
